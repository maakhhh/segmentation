Для запуска необходимо скачать docker, после этого:

1. Закинуть модель unet.keras в папку model/ (потом возможно что-то придумаю с хранилищем для этого файла, но пока он в гит не влазит, берите в тг)
2. В консоли заходим в корневую папку проекта
3. Пишем docker-compose up -d
4. Заходим на localhost:5173 (если вдруг апишка с бэка нужна - она на localhost:8000, управление хранилкой - localhost:9001)
5. Кайфуем
