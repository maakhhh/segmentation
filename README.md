Для запуска необходимо скачать docker, после этого:

1. Создать папку /uploads на уровне с папками backend и frontend (это временно и я не уверен, что это нужно для запуска, но на всякий случай делайте)
2. Закинуть модель unet.keras в папку model/ (потом возможно что-то придумаю с хранилищем для этого файла, но пока он в гит не влазит, берите в тг)
3. В консоли заходим в корневую папку проекта
4. Пишем docker-compose up -d
5. Заходим на localhost:5173 (если вдруг апишка с бэка нужна - она на localhost:8000, управление хранилкой - localhost:9001)
7. Кайфуем
